## 재정의
- 선수가 자신의 바로 앞의 선수를 추월할 때, 추월한 선수의 이름을 부른다ㅏ.
- 경주가 끝났을 때 선수의 이름을 순서대로 출력하기


## 알고리즘
1. 현재 등수대로 선수 배열이 주어진다.
2. 추월한 선수들의 이름을 부른다.
3. 이름이 불린 선수들은 자신의 앞의 선수를 추월한다.
4. 추월당하나 선수는 등수가 하나 밀린다. (swap)
5. 순위가 변동된다.





## 의사코드

선수의 이름
선수의 등수

}

    선수의 인덱스 에서 -n하기
    선수 앞에 있던 경우에는+n하기 


    배열에 넣기
}
