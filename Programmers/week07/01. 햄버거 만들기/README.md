## 재정의
정해진 순서로 햄버거를 포장한다.
- 빵 야채 고기 빵
- 있는 재료에서 햄버거를 몇 개 만들 수 있나.


## 알고리즘
- ingredient 의 재료가 주어진다.
- 순서는 1, 2, 3, 1이어야 한다.
- 재료를 순회하여 1이 나오면 뒤에 재료들이 2, 3, 1이 나오는지 확인한다ㅏ.
- 나오는 경우 해당만큼 제거한다음 다시 시행한다.
- 투 포인터?
