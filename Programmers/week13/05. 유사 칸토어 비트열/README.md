## 재정의 
0과 1사이에 실수로 이루어진 집합으로, [0, 1]부터 시작하여 각 구간을 3등분 해서 가운데 구간을 반복적으로 제외한다.

- 0번째 유사칸토 비트열은 1
- n번째 유사칸토 비트열은 n-1번째 비트열에서의 1을 11011로 치환하고, 0을 00000로 치환하여 만든다
- 남아있는 n 번째 유사칸토어 비트열에서 특정 구간내의 1의 개수 구하기
