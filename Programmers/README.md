## 🙋‍♂️알고리즘 사고 과정
#### 1. 문제 이해
문제의 제약사항 및 입력/출력을 정확히 파악합니다.

#### 2. 익숙한 용어로 재정의
문제를 더 쉽게 이해할 수 있도록 용어와 조건을 단순화하거나 익숙한 방식으로 표현합니다.

#### 3. 어떻게 해결할지 
가능한 알고리즘 또는 자료구조를 선택하고, 해결 방법을 구상합니다.

#### 4. 계획 검증 및 수도 코드 작성
생각한 방법이 맞는지 검토하고, 간단한 수도 코드를 작성합니다.

#### 5. 구현
프로그래밍 언어를 사용하여 구체적인 구현을 합니다.

#### 6. 개선 방법
성능 최적화와 가독성 향상을 고려합니다.
